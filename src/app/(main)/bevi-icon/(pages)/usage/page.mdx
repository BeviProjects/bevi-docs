import { BvIcon } from "bevi-icon";

# Utilizando a biblioteca

## Importação

Após instalar a biblioteca, importe o componente `BvIcon` no arquivo onde deseja utilizá-lo:

```ts
import { BvIcon } from 'bevi-icon'
```

## Uso básico

Para exibir um ícone, basta passar o nome dele pela prop `name`:
```tsx
<BvIcon name='eco' />
```

Sem nenhuma outra configuração, o ícone será renderizado com a variante **solid**, peso **400** e tamanho **32x32**.

---

## Uso avançado

O componente aceita props adicionais para personalização. Você pode combinar variante, peso, tamanho e título de acessibilidade conforme a necessidade do seu projeto:

```tsx
<BvIcon name='eco' variant='duo' width={42} height={42} title='Ícone de folha'/>
```

## Props



| Prop      | Tipo                                    | Padrão    | Obrigatório                              | Descrição                                                  |
| --------- | --------------------------------------- | --------- | ---------------------------------------- | ---------------------------------------------------------- |
| `name`    |  string                                 | —         | <BvIcon name="checkCircle" width={16} /> | O nome do ícone a ser exibido                              |
| `variant` |  'solid' \| 'duo' \| 'dark' \| 'light'  |  'solid'  | <BvIcon name="cancel" width={16} />      | A variante visual do ícone                                 |
| `weight`  |  400 \| 600                             |  400      | <BvIcon name="cancel" width={16} />      | O peso do traço do ícone                                   |
| `width`   |  number                                 |  32       | <BvIcon name="cancel" width={16} />      | A largura do SVG em pixels                                 |
| `height`  |  number                                 |  32       | <BvIcon name="cancel" width={16} />      | A altura do SVG em pixels                                  |
| `title`   |  string                                 | —         | <BvIcon name="cancel" width={16} />      | Insere uma tag `<title>` dentro do SVG para acessibilidade |
